# SmithAI Bug Report Template

Use this template when opening a GitHub issue at: https://github.com/Syntaxful/SmithAI/issues

You can generate a prefilled link in-game with `/smithai report <short description>`, or the plugin will save a copy to `plugins/SmithAI/issue_reports.yml` if the link is too long.

---

[Bug Report] Short summary of the issue

## What I expected

Describe what you expected Smith_AI to do instead.

## Steps to reproduce

1. Spawn Smith_AI with `/smithai spawn`
2. (what did you say or command?)

- External connected: (true / false)
- NPC task busy at the time: (true / false)
- Minecraft server version: (e.g., 1.21.1)

## Logs / screenshots

Paste any relevant server console log lines, screenshots, or error messages here.

## Suggested fix / feature

If you have an idea for how to fix it or what a new/improved LLM behavior should look like, describe it here. This helps the developer prioritize and build the next LLM version.
